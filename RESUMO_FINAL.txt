COMO USAR
================================================================================

1. Compilar:
   cd GeradorFrontendEnterprise
   dotnet build

2. Executar:
   dotnet run

3. Acessar Wizard:
   http://localhost:5000/wizard/step1

4. Seguir 5 Etapas:
   - Selecionar entidade
   - Resolver conflitos
   - Configurar layout
   - Gerar código
   - Baixar ZIP

5. Integrar código em seu projeto
